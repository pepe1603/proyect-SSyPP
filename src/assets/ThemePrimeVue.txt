Algunos Estilos por defecto de Prime vue
: p-listbox: Esta clase se utiliza para el componente de lista desplegable. Proporciona estilos para la lista desplegable, como el tamaño, el color y los bordes.

p-checkbox: Esta clase se utiliza para el componente de casilla de verificación. Proporciona estilos para las casillas de verificación, como el tamaño y los colores.

p-radiobutton: Esta clase se utiliza para el componente de botón de opción. Proporciona estilos para los botones de opción, como el tamaño y los colores.

p-togglebutton: Esta clase se utiliza para el componente de botón de alternancia. Proporciona estilos para los botones de alternancia, como el tamaño y los colores.

p-spinner: Esta clase se utiliza para el componente de spinner. Proporciona estilos para el spinner, como el tamaño y los colores.

p-tooltip: Esta clase se utiliza para el componente de tooltip. Proporciona estilos para el tooltip, como el tamaño, el color y la posición.

p-tabview: Esta clase se utiliza para el componente de pestañas. Proporciona estilos para las pestañas, como el tamaño, el color y los bordes.

p-menu: Esta clase se utiliza para el componente de menú. Proporciona estilos para el menú, como el tamaño, el color y la posición.

p-tree: Esta clase se utiliza para el componente de árbol. Proporciona estilos para el árbol, como el tamaño, el color y los iconos de los nodos.

p-chips: Esta clase se utiliza para el componente de chips. Proporciona estilos para los chips, como el tamaño, el color y los bordes.

p-toolbar: Esta clase se utiliza para el componente de barra de herramientas. Proporciona estilos para la barra de herramientas, como el tamaño, el color y los bordes.
